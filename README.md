Lucky Draw
==========

![Open Source Lucky Draw screenshot](<docs/Screenshot 2025-12-25 Lucky Draw - Open Source.png>)

Open Source Lucky Draw javascript program is developed by open source contributors.

## Local Setup

1. Clone this project to local path.
2. Modify data/data.json to enter your own lucky draw entries (names, numbers, etc.)
3. Browse local index.html to run

## Idea

This projcect is developed by Sammy Fung and other contributors, modified from a previous work by Haggen So.

The theme comes from  Hong Kong Open Source Conference 2016, and this project is used in open source conferences and events.
