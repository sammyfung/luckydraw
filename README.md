Lucky Draw
==========

Open Source Lucky Draw javascript program is developed by open source contributors.

## Local Setup

1. Clone this project
2. Modify data/data.json to enter your own lucky draw entries (names, numbers, etc.)

## Idea

This projcect is developeed by Sammy Fung and other contributors, based on previous prototype by Haggen So. 

The theme comes from  Hong Kong Open Source Conference 2016.
